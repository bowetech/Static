Front-end Boilerplate

Using Sass and Gulp

Here's a quick set-up guide:

  • Install Node
  • Install NPM -Package manager
  • Install Staic Framework dependancy by running "npm install" at the commmand line in your project directory
  • Install the gulp-cli on your command line by running npm install gulp-cli -g
  • Run "gulp" on the command line.

Project Info

This is a starter front-end project which will compile SCSS and JS files using Gulp 4.
